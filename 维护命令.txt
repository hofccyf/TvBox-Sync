1. 🔄 日常同步与日志排查
如果你不想等计划任务触发，想立刻更新数据，或者想看看到底有没有报错：

手动触发一次全量同步：
sh /opt/osc_sync.sh

实时滚动查看同步日志（排错必备）：
tail -f /var/log/osc_sync.log
(按 Ctrl+C 退出查看)

查看全部历史日志：
cat /var/log/osc_sync.log

2. ⚙️ 系统服务与状态管理

在修改了路由器端口、或者感觉 Web 服务卡死时使用：
重启 Web 服务 (uhttpd)：
/etc/init.d/uhttpd restart

重启计划任务服务 (Cron)：
/etc/init.d/cron restart

查看当前已生效的定时任务：
crontab -l | grep osc_sync

重载防火墙（如果在安装时开启了 WAN 访问）：
/etc/init.d/firewall reload

3. 🗑️ 彻底卸载与清理遗留
如果你不想用了，或者想完全推倒重来，按顺序执行以下命令可以将其“连根拔起”，不留任何垃圾：

第一步：清理系统配置（Web 与 防火墙）
uci -q delete uhttpd.osc_vod
uci commit uhttpd
uci -q delete firewall.osc_vod
uci commit firewall

第二步：清理计划任务
crontab -l | grep -v "osc_sync.sh" > /tmp/clean_cron
crontab /tmp/clean_cron
rm -f /tmp/clean_cron

第三步：删除所有实体文件并重启服务
rm -rf /opt/TVBoxOSC
rm -f /opt/tv.m3u
rm -f /opt/index.html
rm -f /opt/osc_sync.sh
rm -f /var/log/osc_sync.log
/etc/init.d/uhttpd restart
/etc/init.d/cron restart
/etc/init.d/firewall reload
